# TradeHub

## Graduation Project

TradeHub is a modern Business Management System developed as a Graduation Project. The application helps business owners manage products, categories, customers, orders, business information, and account settings through a responsive and user-friendly interface.

---

## My Role

I worked as the **Front-End Developer** using **Angular 17**.

### Responsibilities

* Developed the complete Front-End using Angular 17
* Built reusable Standalone Components
* Integrated RESTful APIs
* Implemented JWT Authentication
* Developed Dashboard pages
* Created CRUD operations
* Implemented Routing and Navigation
* Built Responsive UI
* Implemented Form Validation
* Handled API Errors and User Feedback
